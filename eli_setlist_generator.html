How to use:
  • Put the names of the songs in the box labeled "names of the songs."
  • Use the checkboxes to select which songs can transition into which
  • Click "Generate Setlist" to generate a random setlist using only the allowed transitions
  • The songs and all the transitions are encoded into JSON format in the box labeled "songs and transitions JSON code." You can copy it out of there and save it somewhere else. To restore what you had on the page, paste the code back into the "songs and transitions JSON code" box."
